# trasc
trasc is just a little training tool for linux I made for fast debugging in registers for assembly progamming
so feel free to use it for study or changes

to compile you can use gcc with the following command:

gcc -o trasc trasc.c 
then you just have to move the executable to the bin folder of your system in
usr/local/bin/

you can use the command 
mv trasc usr/local/bin
